Project Overview
This project is created as part of the Digital Literacy course. The main aim of this project is to understand the use of digital tools, build a professional online presence, communicate properly in an online environment, and learn about cyber safety.

In this project, I worked as a Student Digital Ambassador and completed different tasks related to digital awareness and practical skills required in everyday life.

Module / Task Summaries
Task 1 – Digital Literacy Awareness
In this task, I created an infographic using Canva to explain digital literacy, its importance and safe internet practices using an online tool.

Task 2 – Digital Portfolio
In this task, I created and updated my profiles on platforms like GitHub, LinkedIn and Kaggle to build a professional digital presence.

Task 3 – Platforms
In this task, I explored coding platforms and completed a beginner-level problem. I also created a Google Form quiz related to digital literacy.

Task 4 – Email Etiquette
In this task, I wrote professional emails and created a checklist for proper social media usage.

Task 5 – Cybercrime Awareness
In this task, I studied a cybercrime case(like UPI Fraud and Phishing) and created a checklist to understand how to stay safe online.
