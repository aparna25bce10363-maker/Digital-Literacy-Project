Task 3 – Platform Notes


Coding Practice Platform

For this task, I used HackerRank as my coding practice platform. I completed a beginner-level problem to understand how coding platforms work. This helped me improve my problem-solving skills and get familiar with writing code in an online environment.

Coding platforms like HackerRank are useful for practicing programming, improving logic and preparing for technical interviews. They provide different types of questions that help in building strong coding skills.

Google Workspace Collaboration

In this part, I created a Google Form based on Digital Literacy Awareness. The form contains 5 questions, including multiple-choice and short answer types. This form can be used to test the basic knowledge of students about digital literacy.

I also viewed the responses using Google Sheets, which helps in organizing and analyzing the data collected from the form.

Learning Outcome

Through this task, I learned how to use coding platforms for practice and how to create and manage online forms using Google Workspace. These tools are very useful for academic work, assignments and collaboration with others.
